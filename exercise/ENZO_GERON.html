  1. Que es un sistema de control de versiones?

    Un sistema de control de versiones es una herramienta que nos permite trabajar en equipo y llevar un historial de todos los cambios que realizamos en nuestro proyecto.

  2. Cual es la diferencia entre git y github?

    Git es la tecnologia que nos permite que trabajar con un sistema de control de versiones y github es una plataforma en la nube que nos permite usar git, junto con la posibilidad de administrar todos nuestros proyectos y complementarlos con herramientas de devops.
